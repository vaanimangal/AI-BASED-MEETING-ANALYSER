# AI-BASED-MEETING-ANALYSER
PointTaker is an AI-powered meeting analyzer that automates  transcription, summarization, sentiment analysis, and action-item extraction.
## Live Demo
https://www.pointtaker.lovable.app
## Features
- AI meeting transcription
- Smart summarization
- Speaker detection
- Action-item extraction
- Meeting analytics dashboard
